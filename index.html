ABOUT

My name is Duncan, a 17-year-old student hailing from the Netherlands. Currently, I am enrolled at the ROC Mondriaan located in The Hague, pursuing a degree in Software Development. I am presently in my second year of the program and actively seeking an internship opportunity for the upcoming academic year. Specifically, I am interested in securing an internship related to web development. My skill set includes proficiency in HTML, CSS, JavaScript, Lua, and PHP. Moreover, I possess expertise in utilizing the Bootstrap framework.

To create my portfolio, I employed a color scheme and integrated Bootstrap. The design of the pages incorporates a sticky display, resulting in overlapping sections. The interaction of project cards is achieved through a carousel, which is modified upon the firing of the 'show.bs.modal' event. This adjustment involves manipulating an element attribute that contains an index. Subsequently, this index is used to access an array containing objects. Each object stores project information, which is then inserted into the respective element. The images in the carousel are used as inspiration.

Four in a row

I created a challenging version of Tic Tac Toe called "Four in a Row" using HTML, CSS, and JavaScript.

Originally designed for Player vs. Player, I added an AI opponent for a higher grade. The computer's moves were randomized, making the game unpredictable and engaging.

The interface was user-friendly, displaying the current player's turn, name and the game result. It was an opportunity to showcase my skills in web development and strategic thinking.
